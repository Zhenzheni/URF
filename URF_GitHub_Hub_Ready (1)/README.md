# The Unified Relational Framework

The Unified Relational Framework (URF) is an independent research programme investigating a geometric parent construction associated with the Reuleaux triangle, its exact incidence structure, and possible higher-dimensional continuations.

This repository separates established geometry, explicitly introduced construction principles, conjectures, and physical hypotheses. It is a working research archive: inclusion here does not by itself indicate that a claim has been proved or empirically validated.

## Paper I — Foundational Geometry

**The Complete Reuleaux Generating Arrangement: Foundational Geometry and the Parent Construction**

The first paper defines the complete generating-circle arrangement, derives its six-point incidence geometry and exact \(2:1\) centroidal ratio, and distinguishes those planar theorems from the conditional tetrahedral extension.

## Claim Status

URF material should be read according to the status assigned to each claim:

1. **Definition** — specifies the object under study.
2. **Theorem** — follows deductively from stated definitions and assumptions.
3. **Construction principle** — introduces an explicit additional rule.
4. **Conjecture** — states a precise proposition not yet proved.
5. **Physical hypothesis** — proposes a connection to nature requiring empirical testing.
